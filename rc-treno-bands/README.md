# RC Treino Bands

Aplicativo inteligente para treinos com elásticos de resistência, peso corporal e equipamentos domésticos.

## Objetivo

Criar um aplicativo simples, moderno e inteligente para organizar treinos realizados em casa.

## Recursos previstos

- Treinos A, B e C
- Biblioteca de exercícios
- Cronômetro de descanso
- Histórico de treinos
- Progressão automática
- Estatísticas
- IA Personal Trainer

## Tecnologias

- React
- Next.js
- TypeScript
- Tailwind CSS
- Firebase
- GitHub
- Lovable
- Google AI Studio
